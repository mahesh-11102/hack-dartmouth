Idea About App

Our Inspiration to create this project started with the thought of how a robo or AI can create music for us. Keeping this in mind we tried to implement this idea in our project. For Acheving this goal we tried to use Genetic Algorithm using python, and also we made a website using HTML , CSS and JAVASCRIPT and also tried to implement API. We also created a login page for our user so they can visit our website and create music for themselves.
